# Zsolt Bedohazi

I am a Computer Science Ph.D. Student (Eötvös Loránd University) and Senior Software Engineer (MSCI Inc.) passionate about using machine learning, deep learning, to solve challenging real world problems especially in the biomedical domain with strong scientific background and experience in research and industry.

In my PhD I focus on computer vision and applied deep learning for biomedical imaging. I currently work on developing and applying supervised and unsupervised deep learning models and methods on pathological whole slide images.


## 🏆 Competition Results

1. ### [Nightingale High Risk Breast Cancer Prediction Contest 1](https://app.nightingalescience.org/contests/3jmp2y128nxd):

    Participated in this international challenge to predict the stage of a patient's cancer using machine learning and deep learning with only slide images generated by breast biopsies.
    Teamed up with [Andras Biricz](https://github.com/abiricz) and [Oz Kilim](https://github.com/ozkilim) to create an efficient algorithm based on deep multi-instance learning and attention mechanism.

    🏆 Won the contest.

Participated with a team of four: [Alex Olar](https://github.com/qbeer), [Andras Biricz](https://github.com/abiricz), [Bendeguz Sulyok](https://github.com/bsulyok).
